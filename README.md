## 🚀 Features

✨ Voice Recognition (Speak and convert to text)  
🌍 Multi-Language Translation (Hindi, English, Bengali, Punjabi, etc.)  
⚡ Fast AI-powered Translation  
🎯 Clean & Modern UI (Google Translate style)  
🔄 Language Swap Feature  
💾 Save Translated Text in Database  
🎨 Gradient Premium Design

## 🧠 How It Works

```text
🎤 User Speaks
   ↓
📝 Speech → Text
   ↓
🌍 Send to Laravel Backend
   ↓
🤖 OpenAI API Translation
   ↓
📦 Save in Database
   ↓
🖥️ Show Result in UI
